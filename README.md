# desafio02

### Requerimentos
* PHP **>=** 5.6
* Composer
* VPN
* Usuário e Senha

### Como utilizar:
* Clone este repositório 
```
git clone https://github.com/haruoSugano/desafio02
```
* Instalar o composer
```
composer install
```
* Navegar até o diretório desafio02, digitar o comando:
```
php example/test.php
```
